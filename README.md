## Environment Variables

### `TM_BIND`

The address and port the server will bind to. Default is `:8080`.

### `TM_DATABASE_URI`

The database connection URI. Example for PostgreSQL: `postgresql://user:password@localhost:5432/dbname?sslmode=disable`.
